# Pong

Classic 2-Player Pong Game

The objective of this game is to try to score until 3. First player to have a score of 3 wins. <br />

Controls: <br />
Player 1 : <br />
  W - move up <br />
  S - move down <br />
  D (Hold) - power up <br />
Player 2 : 
UP keypad - up <br />
DOWN keypad - down <br />
LEFT keypad (HOLD) - power up
